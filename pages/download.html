Frequently Asked Questions

What if I get a "Node.js not found" error?
The installer will automatically check for Node.js and provide a download link if it's not installed. Simply install Node.js from the provided link and run the ChitoNet installer again.
Does the app require .NET 9?
Yes, ChitoNet Control Panel requires .NET 9 Runtime to be installed on your system. The installer will automatically check for .NET 9 and provide a download link if it's not installed. You can also download it directly from Microsoft's official website.
How do I uninstall ChitoNet Command?
You can uninstall by running the uninstall.bat file from the installation folder (C:\Program Files\ChitoNet\), or use Windows Control Panel > Programs and Features.
Can I install on multiple computers?
Yes! Each user can download and install ChitoNet Command on their own computer. Your data is synchronized through the cloud, so you can access it from any installation.
Is an internet connection required?
Yes, an internet connection is required for Firebase authentication, cloud synchronization, and real-time device monitoring.
